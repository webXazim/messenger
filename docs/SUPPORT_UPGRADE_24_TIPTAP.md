# Support Knowledge editor — Tiptap integration

The custom `contenteditable` editor was replaced with Tiptap open-source packages.

Included features:
- headings, paragraphs, bold, italic, underline, and strike
- lists, block quotes, dividers, and code blocks
- secure HTTPS or mailto links
- resizable tables with row and column controls
- undo, redo, clear formatting, focus mode
- word and character counts
- client-side paste cleanup
- server-side Bleach allowlist sanitization before persistence and rendering

The existing Support article CRUD, permissions, author attribution, version history, website availability, and account scoping remain unchanged.
